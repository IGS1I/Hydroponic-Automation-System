# Hydroponic-Automation-System
A space-efficient hydroponic automation system that enables users to grow fresh vegetables at home.

## Components
- ESP 32
- EC/TDS Sensor (sensor module + probe)
- Hilitand PH Module Probe Kit (sensor Module + PH Electrode Probe)
- Ultrasonic sensor
- 3 Signal relays (dual relay is fine to use)
- 12V DC Dosing Pump (Peristaltic Dosing Head with Connector for Arduino) x2
- 120V submersible water pump
- 12-24V Solar/DC water pump
- hydroponics grow bed
- transparent pvc pipes
- Water resevoir
